# "Call-center issues" (service && client)
(CDEK test task)

<b>Техническое задание:</b><br>Сделать сервис для курьера, чтобы добавлять задания в Call-центр.

Для этого нужно сделать:

Web-интерфейс для курьера, где он указывает номер заказа и нажимает кнопку “Не успеваю”.

При нажатии добавляется новое задание для call-центра в БД.

Web-интерфейс для оператора call-центра: Список заданий для прозвона клиентов. Должна
быть таблица с колонками:

Номер заказа, Дата и Время добавления задания.

Также должен быть фильтр или поиск “по номеру заказа”.

****************************************************

<h2>Проект состоит из двух приложений:</h2>

1) service (веб-сервис для получения списка отложенных курьером заказов и добавления новых)

Spring boot

БД: MySQL (параметры подключения в application.properties)

dump БД: issues.sql (в корневом каталоге проекта) 

Сборщик: Maven

<h4>Запуск сервиса:</h4>

Вариант 1: mvn spring-boot:run

Вариант 2: mvn clean package, в IDE - запуск Application.class

<b>URL: http://localhost:9000/postponed</b>

=======================================================================================

2) client (веб-приложение реализующее 2 вида web-интерфейсов: для курьера и для оператора call-центра)

<h4>Запуск клиента:</h4>

Вариант 1: mvn clean package, java -jar target/client.jar

Вариант 2: mvn spring-boot:run

Вариант 3: mvn clean package, в IDE - запуск Application.class

<b>URL: http://localhost:8080/index.html<b>
